# Publicacao do site

Este documento explica como publicar o site pessoal usando GitHub Pages.

## Requisitos

- Conta no GitHub.
- Repositorio com os arquivos do site.
- Arquivo `index.html` na raiz do projeto.

## Passos para publicar no GitHub Pages

1. Envie os arquivos para o repositorio no GitHub.
2. Acesse o repositorio no GitHub.
3. Entre em `Settings`.
4. Abra a secao `Pages`.
5. Em `Build and deployment`, selecione:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` ou `master`
   - `Folder`: `/root`
6. Clique em `Save`.
7. Aguarde o GitHub gerar o endereco do site.

O endereco esperado para este projeto e:

```text
https://pablobretan.github.io/meu-site/
```

## Checklist antes de publicar

- Conferir se `index.html`, `sobre.html` e `contato.html` abrem corretamente.
- Testar os links do menu.
- Testar os links da pagina de contato.
- Verificar se as imagens aparecem.
- Conferir se o site abre bem em celular.
- Confirmar que o ano do rodape esta atualizado.

## Atualizacao do site

Para atualizar o site publicado:

1. Edite os arquivos locais.
2. Teste no navegador.
3. Envie as alteracoes para o GitHub.
4. Aguarde o GitHub Pages atualizar a publicacao.
