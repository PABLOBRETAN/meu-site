# Meu Site de Perfil

Site pessoal de Pablo Rafael Bretan, criado com HTML, CSS, Bootstrap e imagens locais. O projeto apresenta informacoes profissionais, areas de atuacao e canais de contato.

## Paginas

- `index.html`: pagina inicial com resumo do perfil.
- `sobre.html`: apresentacao profissional, areas de atuacao, hobbies e interesses.
- `contato.html`: links para WhatsApp, Instagram, LinkedIn, GitHub, YouTube e e-mail.

## Tecnologias

- HTML5
- CSS3
- Bootstrap 5.3 via CDN
- Google Fonts
- Imagens locais na pasta `imagens/`

## Estrutura do projeto

```text
meu-site/
|-- index.html
|-- sobre.html
|-- contato.html
|-- imagens/
|-- docs/
|-- package.json
`-- README.md
```

## Como abrir localmente

Como o site e estatico, basta abrir o arquivo `index.html` no navegador.

Opcionalmente, use uma extensao como Live Server no VS Code para navegar pelo site com um servidor local.
