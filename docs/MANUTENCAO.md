# Manutencao do site

Este documento organiza boas praticas para manter o site atualizado.

## Atualizar conteudo

Arquivos principais:

- `index.html`: dados principais do perfil.
- `sobre.html`: texto de apresentacao e areas de atuacao.
- `contato.html`: links e canais de contato.

Ao alterar textos, confira se os acentos aparecem corretamente no navegador.

## Atualizar imagens

As imagens ficam em:

```text
imagens/
```

Boas praticas:

- Usar nomes simples, sem espacos.
- Evitar imagens muito pesadas.
- Conferir se o caminho usado no HTML esta correto.
- Sempre preencher o atributo `alt` nas imagens.

## Atualizar links de contato

Na pagina `contato.html`, confira periodicamente:

- WhatsApp
- Instagram
- LinkedIn
- GitHub
- YouTube
- E-mail

## Melhorias futuras sugeridas

- Separar o CSS em um arquivo `style.css`.
- Corrigir textos com caracteres quebrados, caso aparecam no navegador.
- Criar uma pagina de projetos.
- Adicionar metatags de SEO.
- Adicionar favicon.
- Testar acessibilidade e contraste.
